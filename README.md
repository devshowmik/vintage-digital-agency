# 💾 Vintage Web Agency

A retro-inspired web development agency website designed with the nostalgic look and feel of the 90s internet. The project combines modern web development with classic layouts, pixel-style graphics, vintage typography, bold colors, and old-school UI elements.

## 🌐 Live Demo

👉 https://your-live-site.com

## ✨ Features

- 🎨 Authentic 90s-inspired design
- 📱 Fully responsive layout
- ⚡ Fast and lightweight
- 🖥️ Retro UI elements and animations
- 💼 Agency landing page
- 📞 Contact section

## 🛠️ Built With

- HTML5
- CSS3
- Tailwindcss

## 🚀 Getting Started

Clone the repository:

```bash
git clone git https://github.com/devshowmik/vintage-digital-agency.git
```

Open the project folder and launch `index.html` in your browser.
